Интерфейс умного дома

Для платформы iRidium mobile

Информация о проекте

Перед нами стояла задача создания дизайна интерфейса приложения для управления умным домом с мобильных устройств.
Заказчиком была выбрана стилистика Metro.
Разработанный UI в дальнейшем был внедрен как базовый интерфейс элементов в iRidium GUI Editor (ПО проектирования системы для конкретной конфигурации умного дома).
