Дизайн системы мониторинга транспорта

Задача

  Разработать дизайн пользовательского интерфейса десктопного Windows приложения для системы мониторинга транспорта.

Концепт

  Дизайн разработан для WPF приложения с использованием пакета элементов DevExpress.
  В основе приложения лежит Docking Windows интерфейс.
  Стилистика дизайна базируется на дизайне MacOS 2010 - 2012, используя для акцентов оранжевые цвета.
  Из-за высокой оценки времени, необходимого для интеграции данного дизайна в проект (6 месяцев), заказчиком было принято решение не менять стандартный UI DevExpress. Концепт не был завершен.
